# K-Meal Bridge 🍜

> A delivery assistant service for foreigners in Korea  
> 외국인을 위한 한국 배달앱 주문 대행 서비스

---

## 📌 프로젝트 개요

**K-Meal Bridge**는 한국에 거주하는 외국인들이 언어와 결제 장벽 없이 한국 배달 서비스를 이용할 수 있도록 돕는 주문 대행 플랫폼입니다.

### 🎯 프로젝트 목적

1. **언어 장벽 해소**: 영어 기반 인터페이스로 한국 음식 주문 간소화
2. **결제 편의성**: 외국인 친화적 결제 시스템 제공
3. **문화적 접근성**: 한국 배달 문화 및 음식에 대한 가이드 제공
4. **서비스 연결**: 한국 배달앱과 외국인 사용자를 연결하는 브릿지 역할

---

## 🌍 타깃 사용자

- 🎓 한국 거주 외국인 유학생
- 💼 외국인 근로자 및 주재원
- ✈️ 한국 방문 여행자
- 🌏 한국 배달 서비스에 관심 있는 모든 외국인

**예상 사용자 규모**: 국내 체류 외국인 약 250만명 + 연간 관광객 1,100만명

---

## ✨ 주요 기능

### Phase 1: MVP (현재 개발 단계)

- ✅ 다국어 지원 (영어 우선)
- ✅ 소셜 로그인 (Google, Kakao)
- ✅ 음식점 검색 및 메뉴 탐색
- ✅ 주문 대행 시스템
- ✅ 실시간 주문 상태 추적
- ✅ 알림 시스템 (Email, Kakao)

### Phase 2: 고도화 (향후 계획)

- 🔄 추가 언어 지원 (중국어, 태국어, 베트남어)
- 🤖 AI 기반 메뉴 추천
- 🔔 실시간 배달 추적
- 👥 사용자 커뮤니티 (리뷰, 평점)
- 📱 모바일 앱 개발
